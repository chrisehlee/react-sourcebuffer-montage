# react-sourcebuffer-montage example

## install

```
$ git clone git@github.com:slopen/react-sourcebuffer-montage.git
$ cd react-sourcebuffer-montage/example
$ npm i
```

## bootstrap videos

```
$ cd data && mkdir videos
$ chmod 777 *.sh
$ sh bootstrap.sh
```

## start

```
$ npm start
```
